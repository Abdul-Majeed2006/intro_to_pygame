# PyGame Fundamentals: An Engineering-First Curriculum

**Master the engineering foundations of 2D game development by architecting professional-grade systems from scratch.**

## Focus & Discipline
This curriculum is a transition from writing Python scripts to architecting game engines. It eliminates common beginner shortcuts and teaches industry-standard patterns like Object-Oriented Sprites, State Machines, and Deterministic Loops.

## Audience & Exclusions
- **Target**: Python learners who want to understand real-time system architecture.
- **Excluded**: Learners seeking "quick games," copy-pasted solutions, or shallow tutorials.
